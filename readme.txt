== Description ==
LR Faq to the admin panel which allows you to show your faq on your website the easy with deffernt styles

You can quickly add faq to your faq(s), display them anywhere with a easy shortcode.
it’s totally free.

Faq showcase is available in five styles.
For more features Please contact us at [Logicrays](https://www.logicrays.com/contact-us/)
If you like this plugin Please rate us :)

== Plugin Features ==
Responsive faq plugin
All modren browser support
works with anytype of theme
Easy to customize for developeres
Easy to configure

== Shortcode ==

For All Faq
`
[LRFAQ]
`
Shortcode for number of faq
`
[LRFAQ limit="5"]
`

== Installation ==

1. Upload Zip file lr-faq to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
